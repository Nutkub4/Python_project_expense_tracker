# Expense Tracker

Python application for tracking and managing personal expenses.

## Features
- ✅ Add expenses with amount, category, date
- ✅ View all expenses with total
- ✅ Edit existing expenses
- ✅ Delete expenses
- ✅ Save data to JSON file
- ✅ Load previous expenses

## How to Run
```bash
python Python_project_expense_tracker.py
```

## Requirements
- Python 3.6 or higher
- No external libraries needed

## Project Structure
expense-tracker/
├── expense.py       # Main program
├── expenses.json    # Data storage (auto-created)
└── README.md       # Documentation

## Object-Oriented Programming
- **Expense class** with private attributes
- **Encapsulation** using getters and setters
- **Data persistence** with JSON
